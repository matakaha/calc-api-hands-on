# 機能要件

## 1. 概要

本プロジェクトは、HTTP リクエストを受け付け、掛け算および割り算を実行する 2 つの計算 API を提供します。

---

## 2. API 一覧

| API名 | エンドポイント | 説明 |
|-------|---------------|------|
| 掛け算 API | `/multiply` | A × B の計算結果を返す |
| 割り算 API | `/divide` | A ÷ B の商と余りを返す |

---

## 3. 入力仕様

### 3.1 リクエスト形式

- **メソッド**: GET
- **パラメータ形式**: クエリストリング

### 3.2 パラメータ

| パラメータ名 | 必須 | 型 | 説明 |
|-------------|------|-----|------|
| A | ○ | 整数 | 計算対象の値1 |
| B | ○ | 整数 | 計算対象の値2 |

### 3.3 入力値の制約

| 項目 | 制約 |
|------|------|
| 許可される値 | 0以上の整数（非負整数） |
| 桁数制限 | 9桁以内（0 ～ 999,999,999） |
| 負の数 | 不可（エラー） |
| 小数 | 不可（エラー） |

---

## 4. 出力仕様

### 4.1 レスポンス形式

- **Content-Type**: text/plain
- **文字エンコーディング**: UTF-8
- **HTTPステータスコード**: 常に 200（エラー時も含む）

### 4.2 掛け算 API (`/multiply`)

計算結果を数値として返却します。

**リクエスト例**:
```
/multiply?A=3&B=5
```

**レスポンス例**:
```
15
```

### 4.3 割り算 API (`/divide`)

商と余りを返却します。

**リクエスト例**:
```
/divide?A=5&B=2
```

**レスポンス例**:
```
商: 2, 余り: 1
```

---

## 5. エラー仕様

### 5.1 エラーレスポンス形式

- **HTTPステータスコード**: 200
- **Content-Type**: text/plain
- **言語**: 日本語

### 5.2 エラーケース一覧

| エラー条件 | エラーメッセージ例 |
|-----------|------------------|
| パラメータ A が未指定 | パラメータ A が指定されていません |
| パラメータ B が未指定 | パラメータ B が指定されていません |
| A が整数でない | パラメータ A は整数で指定してください |
| B が整数でない | パラメータ B は整数で指定してください |
| A が負の数 | パラメータ A は0以上の整数で指定してください |
| B が負の数 | パラメータ B は0以上の整数で指定してください |
| A が9桁を超える | パラメータ A は9桁以内で指定してください |
| B が9桁を超える | パラメータ B は9桁以内で指定してください |
| B が0（割り算のみ） | 0での除算はできません |
| 計算結果が9桁を超える | 計算結果が9桁を超えています |

---

## 6. 使用例

### 6.1 掛け算 API

**正常系**:
```
リクエスト: /multiply?A=123&B=456
レスポンス: 56088
```

**エラー系**:
```
リクエスト: /multiply?A=1000000000&B=2
レスポンス: パラメータ A は9桁以内で指定してください
```

### 6.2 割り算 API

**正常系**:
```
リクエスト: /divide?A=100&B=7
レスポンス: 商: 14, 余り: 2
```

**エラー系**:
```
リクエスト: /divide?A=10&B=0
レスポンス: 0での除算はできません
```
